🔑 GitHub 토큰 설정
Personal Access Token (PAT)

GitHub에 파일을 저장하려면 토큰이 필요합니다. 토큰은 이 브라우저 세션에만 저장되며, 페이지를 닫으면 사라집니다.

토큰이 설정되지 않았습니다.

PAT 발급 방법 보기
  1. GitHub.com 로그인 후 오른쪽 상단 프로필 → Settings 클릭
  2. 왼쪽 하단 Developer settings 클릭
  3. Personal access tokens → Tokens (classic)
  4. Generate new token (classic) 클릭
  5. Note: 적당한 이름 입력, Expiration: 원하는 기간 설정
  6. 권한에서 repo 체크박스 체크
  7. Generate token 클릭 후 생성된 토큰 복사
